All successful responses follow this envelope:

```json
{
  "response_type": "success",
  "description": "Human-readable result summary",
  "data": { },
  "warnings": []
}
```

| Field | Type | Description |
|-------|------|-------------|
| `response_type` | string | `"success"` or `"error"` |
| `description` | string | Human-readable summary |
| `data` | object or null | Response payload |
| `warnings` | array or null | Access warnings, if any (see [AHJ Access Control](/concepts/ahj-access)) |
