## Key capabilities

- **AHJ data** — List, search, and retrieve detail for Authorities Having Jurisdiction scoped to your subscription
- **Data points** — Query code data point values with filtering by AHJ, state, data point type, and trade group
- **Reports** — Generate comparison matrices and CSV exports across AHJs and data point types
- **Collections** — Execute saved data pull configurations against any set of AHJs
- **Reference data** — Look up trade groups and units of measure used across the platform
