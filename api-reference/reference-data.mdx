## List trade groups

<Snippet file="auth-header.mdx" />

```bash
GET /v1/trade-groups
```

Returns a paginated list of trade groups. Trade groups classify data point types into professional disciplines (e.g. Electrical, Plumbing, Mechanical).

### Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `page` | integer | `1` | Page number |
| `page_size` | integer | `100` | Items per page (max 500) |

### Example request

```bash
curl -X GET "https://connect.fordje.com/v1/trade-groups?page_size=5" \
  -H "X-API-Key: your-api-key"
```

### Example response

```json
{
  "response_type": "success",
  "description": "Retrieved 5 trade group(s)",
  "data": {
    "items": [
      { "id": 1, "name": "Electrical" },
      { "id": 2, "name": "Plumbing" },
      { "id": 3, "name": "Mechanical" },
      { "id": 4, "name": "Building" },
      { "id": 5, "name": "Fire" }
    ],
    "pagination": {
      "page": 1,
      "page_size": 5,
      "total": 8,
      "total_pages": 2
    }
  }
}
```

## List units of measure

```bash
GET /v1/units-of-measure
```

Returns a paginated list of units of measure used for data point values.

### Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `page` | integer | `1` | Page number |
| `page_size` | integer | `100` | Items per page (max 500) |

### Example request

```bash
curl -X GET "https://connect.fordje.com/v1/units-of-measure?page_size=5" \
  -H "X-API-Key: your-api-key"
```

### Example response

```json
{
  "response_type": "success",
  "description": "Retrieved 5 unit(s) of measure",
  "data": {
    "items": [
      { "id": 1, "name": "Boolean", "abbreviation": "bool" },
      { "id": 2, "name": "Text", "abbreviation": "text" },
      { "id": 3, "name": "Numeric", "abbreviation": "num" },
      { "id": 4, "name": "Currency (USD)", "abbreviation": "usd" },
      { "id": 5, "name": "Percentage", "abbreviation": "pct" }
    ],
    "pagination": {
      "page": 1,
      "page_size": 5,
      "total": 12,
      "total_pages": 3
    }
  }
}
```
