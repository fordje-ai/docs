If you run into problems or have questions about Fordje Connect, here is how to get help.

## Contact support

Email **[support@fordje.com](mailto:support@fordje.com)** for:

- Questions about your API key or subscription
- Reporting bugs or unexpected API behavior
- Requesting access to additional AHJs
- Requesting higher rate limits
- General questions about data coverage or accuracy

We aim to respond to support requests within **one business day**.

## Reporting bugs

When reporting a bug, include the following information to help us investigate quickly:

- The **endpoint** you called (e.g., `GET /v1/data-points`)
- The **full request** (URL, headers, query parameters, request body) — redact your API key
- The **full response** (status code, headers, response body)
- The **expected behavior** vs. what actually happened
- The **timestamp** of the request (useful for correlating with our logs)

## Feature requests

We welcome feature requests. Email [support@fordje.com](mailto:support@fordje.com) with:

- A description of the use case you are trying to solve
- How you are currently working around the limitation (if applicable)
- Which endpoints or data you wish were available
