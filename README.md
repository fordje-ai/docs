# Fordje Connect Documentation

[![Docs](https://img.shields.io/badge/docs-docs.fordje.com-blue)](https://docs.fordje.com)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Developer documentation for **Fordje Connect** — a RESTful API for accessing Authority Having Jurisdiction (AHJ) code data.

## What's in the docs

- **Getting Started** — Introduction, quickstart guide, and authentication
- **Core Concepts** — AHJ access control, rate limiting, and error handling
- **API Reference** — Endpoints for AHJs, data points, reference data, reports, and collections

## Quick links

- [Quickstart](https://docs.fordje.com/quickstart)
- [Authentication](https://docs.fordje.com/authentication)
- [API Reference](https://docs.fordje.com/api-reference/introduction)

## Local development

Preview the docs site locally:

```bash
npm i -g mint
mint dev
```

Then open [http://localhost:3000](http://localhost:3000).
