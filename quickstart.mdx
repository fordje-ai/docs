## Prerequisites

You need an API key from your organization's Fordje admin portal. If you don't have one, contact your organization administrator.

## Step 1: Make your first request

Use your API key to list the AHJs your organization has access to.

<CodeGroup>

```bash cURL
curl -X GET "https://connect.fordje.com/v1/ahjs?page=1&page_size=10" \
  -H "X-API-Key: your-api-key"
```

```python Python
import requests

response = requests.get(
    "https://connect.fordje.com/v1/ahjs",
    headers={"X-API-Key": "your-api-key"},
    params={"page": 1, "page_size": 10},
)
data = response.json()
print(data)
```

```javascript JavaScript
const response = await fetch(
  "https://connect.fordje.com/v1/ahjs?page=1&page_size=10",
  {
    headers: { "X-API-Key": "your-api-key" },
  }
);
const data = await response.json();
console.log(data);
```

</CodeGroup>

You'll receive a response like this:

```json
{
  "response_type": "success",
  "description": "Retrieved 10 AHJ(s)",
  "data": {
    "items": [
      {
        "id": 1,
        "name": "City of Los Angeles",
        "type": "city",
        "state_code": "CA"
      }
    ],
    "pagination": {
      "page": 1,
      "page_size": 10,
      "total": 142,
      "total_pages": 15
    }
  }
}
```

## Step 2: Explore data points

Query data point values for specific AHJs. Data points represent code requirements tracked by Fordje.

<CodeGroup>

```bash cURL
curl -X GET "https://connect.fordje.com/v1/data-points?ahj_ids=1,2&page_size=5" \
  -H "X-API-Key: your-api-key"
```

```python Python
response = requests.get(
    "https://connect.fordje.com/v1/data-points",
    headers={"X-API-Key": "your-api-key"},
    params={"ahj_ids": "1,2", "page_size": 5},
)
```

</CodeGroup>

## Step 3: Generate a report

Create a comparison matrix across AHJs and data point types.

<CodeGroup>

```bash cURL
curl -X POST "https://connect.fordje.com/v1/reports/matrix" \
  -H "X-API-Key: your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "ahj_ids": [1, 2, 3],
    "data_point_type_ids": [10, 20, 30]
  }'
```

```python Python
response = requests.post(
    "https://connect.fordje.com/v1/reports/matrix",
    headers={
        "X-API-Key": "your-api-key",
        "Content-Type": "application/json",
    },
    json={
        "ahj_ids": [1, 2, 3],
        "data_point_type_ids": [10, 20, 30],
    },
)
```

</CodeGroup>
